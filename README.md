# TP Bases de données NoSQL : MongoDB

Ce dépôt contient l'ensemble des livrables pour le TP d'introduction à MongoDB.

## Contenu du dépôt

- **Rapport_TP_MongoDB.pdf** : Le rapport complet détaillant l'installation, les requêtes CRUD (Exercice 1) et les requêtes avancées ainsi que Map-Reduce sur la base `gym` (Exercice 2).
- **Rapport_TP_MongoDB.pdf** : Le code source Markdown du rapport.
- **export_db/info_produits.json** : L'export au format JSON de la collection `produits` créée lors de l'Exercice 1.
- **Images (.png)** : Les captures d'écran du terminal validant l'exécution correcte des requêtes de l'Exercice 1.

## Environnement
- MongoDB 8.2.5
- mongosh
- MongoDB Compass
- Robo3T (pour la base gym)
